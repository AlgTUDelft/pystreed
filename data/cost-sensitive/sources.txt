When reusing the datasets in this repository, please refer to the sources as mentioned below:

The UCI Machine Learning repository
Dua, D. and Graff, C. (2019). UCI Machine Learning Repository [http://archive.ics.uci.edu/ml]. Irvine, CA: University of California, School of Information and Computer Science.

Costs are from
Lomax, S. (2013). Cost-sensitive decision tree learning using a multi-armed bandit framework. Phd Thesis. University of Salford.

Note that for some of the datasets, the referenced UCI dataset does not have the same number of instances as in our repository. We follow the dataset versions as described by Lomax, S. (2013), and imported them from https://github.com/shanigu/CostSensitive.
